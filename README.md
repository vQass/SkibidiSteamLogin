# SkibidiSteamLogin

[![NuGet](https://img.shields.io/nuget/v/SkibidiSteamLogin.Core?style=flat-square)](https://www.nuget.org/packages/SkibidiSteamLogin.Core)
[![License: MIT](https://img.shields.io/badge/license-MIT-blue.svg?style=flat-square)](LICENSE)

**SkibidiSteamLogin** is a lightweight .NET library for logging into Steam accounts using the official Steam Mobile authentication system. It simplifies the login process and handles SteamGuard two-factor authentication (2FA).

---

## 🚀 Features

- ✅ Login with username and password  
- 🔒 Support for SteamGuard 2FA codes (via email or mobile authenticator)  
- 🔁 Session management and cookie persistence 
- 📎 Example usage included in the `Guideline` project
